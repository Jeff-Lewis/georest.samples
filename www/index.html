Sheboygan MAP
View Sheboygan MapGuide images and KML using GeoREST. Using OpenLayers.GeoREST you can edit Building features geometry or Parcels attributes.
For example to work, you need to install MapGuide 2.1 on same machine as GeoREST and you need to import in MapGuide Sheboygan sample package.
Download MapGuide Server and Sheboygan sample package from http://www.osgeo.org/mapguide
Note: For feature edit to work you need to change Sheboygan sample package. Change SDF connection READ ONLY property to false for Building and Property Data Source.
